## Prerequisites

- [NodeJs](https://nodejs.org/en/download/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/)

## Installing

```bash

  # Clone repository
  $ git clone https://github.com/pereiradilson/luminosity.git

  # Enter folder
  $ cd luminosity

  # Install dependences
  $ yarn

  #Execute
  $ yarn start

```
